## 📋 О проекте
**GoChat** —  это многопользовательский консольный чат, написанный на Go. Проект демонстрирует принципы сетевого программирования, многопоточности и работы с протоколом TCP.

### Что умеет программа

- Несколько пользователей могут подключаться к серверу одновременно
- Поддержка множества комнат (можно создавать новые и переключаться между ними)
- Защита комнат паролем (приватные комнаты)
- Уведомления о входе/выходе пользователей
- Цветной вывод логов в консоль и запись в файл
- Настройка параметров через конфигурационный файл (порт, таймауты, лимиты)

### Как это работает под капотом

- **Сеть:** TCP — постоянное соединение, сервер может отправлять сообщения клиентам в любой момент
- **Многопоточность:** каждый клиент обрабатывается в отдельной горутине
- **Синхронизация:** мьютексы (`sync.RWMutex`) защищают общие данные от data race
- **Логирование:** три уровня: INFO, WARNING, ERROR


## 📟 Команды пользователя

| Команда | Аргументы | Описание | Пример |
|---------|-----------|----------|--------|
| `/help` | — | Показать список всех команд | `/help` |
| `/rooms` | — | Показать список доступных комнат | `/rooms` |
| `/create` | `<название> [пароль]` | Создать новую комнату (опционально с паролем) | `/create gamers`<br>`/create vip mypass123` |
| `/join` | `<название> [пароль]` | Войти в комнату (с паролем, если требуется) | `/join gamers`<br>`/join vip mypass123` |
| `/leave` | — | Покинуть текущую комнату и вернуться в lobby | `/leave` |
| `/exit` | — | Выйти из чата | `/exit` |


## Гайд по установке

   - https://github.com/amirka7777/GoChat

• Запуск клиента (в другом терминале):
   - make client NAME=YourName
